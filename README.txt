-----------Setting up the Frontend Application in local server---------

Requirements
  -- NPM(Node Package Manager)
  -- Angular/Angular CLI
  
1. Open terminal on O2O-Test-FrontEnd and type the command "npm install",
   to download and install dependancies.
   
2. Then type "ng serve" or "npm start" to run the application(default port http://127.0.0.1:4200)
